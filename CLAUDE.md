# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this repo is

A single-page static personal entity site for Frank Meltke (`meltke.com`), deployed via GitHub Pages from the `main` branch root. There is no build system, no package manager, and no test suite — all code lives in `index.html`.

## Deployment

Push to `main`. GitHub Pages serves directly from the repo root. The `CNAME` file sets the custom domain `meltke.com`.

## CI workflow

`.github/workflows/indexnow-submit.yml` runs on every push to `main` that touches an `.html` or `.htm` file. It:
1. Regenerates `sitemap.xml` from all HTML files in the repo (auto-committed back to `main` via `GH_PAT`).
2. Submits changed URLs to IndexNow using the `INDEXNOW_KEY` secret.

Required GitHub secrets: `INDEXNOW_KEY` (`60c9437e711e49a7b6edea770cc55f90`) and `GH_PAT`.

The IndexNow key file `60c9437e711e49a7b6edea770cc55f90.txt` must stay in the repo root.

## SEO-critical elements

- **Schema.org `Person`** — the `<script type="application/ld+json">` block in `<head>` is the primary structured-data signal for Google's Knowledge Graph. Keep all fields accurate.
- **`sameAs` array** — links to Wikidata (Q139680598), ORCID, LinkedIn, Google Scholar, Google KG (`/g/11rqtzq613`), and `contraco.net/about.html`. These cross-references establish entity authority.
- **`hreflang` alternates** — cross-references to `contraco.de`, `contraco.ru`, `contraco.co.kr` for international SEO.
- **Google Analytics** — tag ID `G-E95SRPTPNX`.
- **Goatcounter** — privacy-friendly counter at `https://meltke.goatcounter.com/count`.
- **`sitemap.xml`** — regenerated by CI; do not edit manually (it will be overwritten on next push).

## Assets

Favicon and icon assets (all `.png` sizes plus `favicon.ico`, `favicon.svg`) are shared with the contraco brand. `Logo_rectangle.png` is a local fallback; the header loads the SVG from `contraco.net` with a JS retry-and-fallback mechanism.
